DMMアフィリエイトのHTMLリンクから、MDX用の `DmmCardImage` と `DmmCard` コンポーネントを生成してください。

## 入力

`$ARGUMENTS` にDMMアフィリエイトのHTMLが含まれている場合はそれを使用してください。
引数が空の場合は、ユーザーにDMMアフィリエイトのHTMLを貼り付けるよう求めてください。

## 手順

1. HTMLから以下の情報を抽出する
   - **href**: 商品ページへのアフィリエイトリンクURL（`<a href="...">` の最初のもの）
   - **imgSrc**: 商品画像URL（`<img src="...">` の最初のもの）
   - **title**: 商品タイトル（`<span>` のテキストコンテンツ）
   - **price**: 価格テキスト（`<p>` の価格部分。「円」までを含めた `〇〇円` 形式にする）

2. 以下のMDXコンポーネントを生成する

```mdx
<DmmCardImage
  href="【href】"
  imgSrc="【imgSrc】"
  title="【商品タイトルの短縮版（製品名のみ、30文字程度）】"
/>

<DmmCard
  href="【href】"
  imgSrc="【imgSrc】"
  title="【商品タイトルのフルテキスト】"
  price="【price】"
/>
```

3. 生成したコードをユーザーに提示する

4. 挿入先のMDXファイルを確認する
   - IDEで開いているファイルがある場合はそれを候補として提示する
   - ファイルが特定できない場合はユーザーに確認する

5. 承認されたら以下の位置にコンポーネントを挿入する
   - **`DmmCardImage`**: フロントマター（`---` で囲まれたブロック）直後の本文先頭
   - **`DmmCard`**: 記事の末尾（最後の行の**直後**、ファイル末尾）

## コンポーネントの仕様

- `DmmCardImage`: 商品画像のみのリンク表示。フロントマター直後の本文先頭に挿入する
- `DmmCard`: 画像・タイトル・価格・DMMリンクを含むカード。締めの一文の**直後**（ファイル末尾）に挿入する
- 両コンポーネントとも `src/components/mdx/DmmCard.tsx` で定義済み

## 注意事項

- `title` の短縮版は製品名・シリーズ名が伝わる範囲で簡潔にする
- URLはそのまま使用し、エンコードを変更しない
- 価格は抽出時点の値であることをユーザーに伝える
